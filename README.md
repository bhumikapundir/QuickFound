# QuickFound
Smart Lost and Found System
in backend terminal run: npm install express mongoose dotenv cors
then run : node server.js



//FRONTEND DEPENDENCIES
We are using:
Tailwind v4
@tailwindcss/postcss
Next.js 16
Versions are locked in package.json

📌Setup Rules (MANDATORY)
After cloning the repo:
in terminal run:
cd frontend
npm install
npm run dev
❌ Do NOT install Tailwind manually
